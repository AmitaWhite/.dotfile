### 다운로드가 필요한 명령어 (macOS 기준)

1. lsd

   - description : 현대적인 ls
   - 사용(aliased) : ls , lt, la, lls 명령어
   - install : `brew install lsd`

2. lolcat
   - description : 예쁘게 해주는 cat ( ascii 문자열)
   - 사용(aliased) : lcat
   - install : `brew install lolcat`
   - add : `lol -a` 로 인해 , line by line 으로 출력되게 해 놓음
